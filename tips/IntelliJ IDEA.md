# File Header

File | Settings | Editor | File and Code Templates

1, 首先点击右上角的File
2, 点击进入Settings
3, 选择 Editor—> File and Code Templates—>includes—>FileHeader
4, 输入签名模板

/**
 * @author  dingxl
 * @date  ${DATE} ${TIME}
 */

注:

1，修改自己的名字
2， ${DATE} ${TIME} 要大写才能获得日期和时间
